rules of the game

Hi, welcome to the "Minefield" game! The goal of the game is to find all the cells that do not contain a bomb, without selecting the ones that do.

Let me explain how it works: the grid has 100 cells, of which 16 contain a bomb. The game starts with all cells covered and you have to select one cell at a time. If you choose a cell that does not contain a bomb, the cell will be uncovered and the number of bombs present in the 8 adjacent cells (if any) will be shown.

If instead you select a cell that contains a bomb, the game ends immediately and you lose. But don't worry, you can always start again!

You can mark the cells that you think contain a bomb with a flag, to help you keep track of the ones you have not yet selected. Please note that the flag does not guarantee the presence of a bomb, but it can be a useful aid.

Remember that you can abandon the game at any time, and that you are responsible for your behavior and any damage caused by the game. Additionally, you cannot use any software or tool that could alter the game or influence the results.

I hope you enjoy playing "Minefield"!
